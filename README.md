# OpenSalve

## Setup

After cloning the repo,

* Install [nodeenv](https://github.com/ekalinin/nodeenv#install)
* Setup environment :
  ```bash
  nodeenv --node=10.9.0 --requirement=requirements.txt env
  ```
* Activate environment :
  ```bash
  . env/bin/activate
  ```
* Install dependencies :
  ```bash
  npm install
  ```
* Start server :
  ```bash
  npm start
  ```
